Pix2pi FAZ 4D-7

Oto Yedek Parça UI

Bu yüzey OEM numarası, eşdeğer parça, muadil parça ve araç uyum bilgilerini core product modelinden ayrı tutmak için hazırlanmıştır. Core ERP ürün modeli sade kalır; oto yedek parça bilgileri extension ve compatibility katmanında izlenir.

01

OEM Numarası

OEM alanı core product içine gömülmez. Ürüne bağlı extension alanı olarak tutulur ve arama kalitesini artırır.

02

Eşdeğer Parça

Aynı işi gören parçalar equivalent group mantığıyla ilişkilendirilir. Böylece muadil parça önerisi yapılabilir.

03

Muadil Parça

Farklı marka veya üreticiden gelen uyumlu parçalar ayrı ilişki modeliyle gösterilir.

04

Araç Uyum

Marka, model, yıl, motor ve kasa bilgileri compatibility modeli olarak tutulur.

05

Stok Durumu

Parça stok bilgisi ERP core stok modeliyle bağlı kalır. Uyumluluk bilgisi stok modelini bozmaz.

06

Tenant-safe Görünüm

Pilot işletme yalnızca kendi tenant verisini görür. Başka tenant parçaları UI yüzeyine sızmaz.
