FAZ 5-19.4 / Internal Preview / API Key Creation Disabled

Pix2pi API Key Yönetim Ekranı

Bu ekran API key yönetim yüzeyi için internal preview iskeletidir. Gerçek API key üretimi, secret reveal, rotation ve canlı sandbox erişimi bu fazda kapalıdır.

Tenant-safe zorunluluklar: X-Tenant-ID, developer account, role guard, permission scope ve audit trail.

Key Inventory

Key adı, masked prefix, tenant scope ve durum bilgisi preview.

Create Disabled

Gerçek API key oluşturma kapalı: api_key_creation_enabled=false.

Masked Secret

Secret değerleri gösterilmez; sadece masked display standardı uygulanır.

Rotation Preview

Rotation akışı preview seviyesindedir; canlı işlem kapalıdır.

Revoke Preview

Revoke akışı audit trail ile tasarlanır; canlı aksiyon sonraki kapılara bağlıdır.

Permission Scope

Read/write scope, webhook scope, sandbox scope ayrımları.

Tenant Scope

Her key tenant sınırıyla ilişkilidir; cross-tenant erişim kapalıdır.

Audit Trail

Key lifecycle aksiyonları audit zorunluluğu ile izlenir.

Security Policy

Expiry, rate limit, secret reveal block ve güvenlik notları.

Launch Guard: production_screen_published=false, real_developer_access_enabled=false, api_key_creation_enabled=false, api_key_reveal_enabled=false, api_key_rotation_enabled=false, sandbox_live_enabled=false.
