Stabilizasyon Kapıları

Green Signals

  • HTTP 200
  • content check PASS
  • nginx -t PASS
  • 4D-11 report PASS
  • tenant security PASS

Warning Signals

  • HEAD/GET farkı
  • cache belirsizliği
  • log görünürlüğü zayıf
  • feedback kanalı eksik
  • rollback kontrolü bekliyor

No-Go Signals

  • cross-tenant sızıntı
  • auth bypass
  • veri bozulması
  • secret sızıntısı
  • nginx config fail
Pilot karar notu: Bu yüzey production Grafana veya Alertmanager alternatifi değildir. Kontrollü pilot stabilizasyon için karar ve görünürlük yüzeyidir. Gerçek production SRE kapısı sonraki fazlarda derinleştirilir.
